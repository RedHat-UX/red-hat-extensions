# Grid

## Usage
To create simple grid layouts within content.  Not intended for use of full page layouts.

```html

  <div>Grid Item 1</div>
  <div>Grid Item 2</div>
  <div>Grid Item 3</div>
  <div>Grid Item 4</div>
  <div>Grid Item 5</div>
  <div>Grid Item 6</div>
  <div>Grid Item 7</div>
  <div>Grid Item 8</div>
  <div>Grid Item 9</div>
  <div>Grid Item 10</div>
  <div>Grid Item 11</div>
  <div>Grid Item 12</div>
</rhx-grid>
```
